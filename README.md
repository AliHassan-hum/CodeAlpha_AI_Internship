# CodeAlpha Artificial Intelligence Internship Tasks

This repository contains all 4 projects completed during my Artificial Intelligence internship at CodeAlpha. Each project is built using Python and features a modern user interface.

## 🛠️ Tasks Developed:

### 1. Language Translation Tool
* **File:** `translator.py`
* **Description:** Translates text from English into Spanish, French, or Urdu. It features a graphical user interface built with CustomTkinter.

### 2. AI FAQ Chatbot
* **File:** `chatbot.py`
* **Description:** A smart chatbot that handles predefined user queries about the CodeAlpha internship using automated matching logic.

### 3. AI Music Generator
* **File:** `music_gen.py`
* **Description:** An algorithmic music composer that generates musical notes and chord sequences, exporting them directly into playable MIDI audio files.

### 4. Real-time Object Detection
* **File:** `object_detection.py`
* **Description:** A computer vision application that utilizes the state-of-the-art YOLOv8 neural network model to detect everyday objects and persons via a live webcam feed.
